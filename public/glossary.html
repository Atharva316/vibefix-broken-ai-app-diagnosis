VibeFix Glossary for Vibe Coders

Plain-English definitions for founders debugging AI-built apps.

Fix-break loop

When an AI tool fixes one issue but creates another bug somewhere else.

Rollback vs fix-forward

Rollback means returning to the last working version. Fix-forward means keeping the current version and applying the smallest safe repair.

RLS (Row Level Security)

Supabase rules that decide which rows a user can read or write. Bad RLS policies often make data disappear even when the app UI looks correct.

Environment variables

Secret or config values used by your app, like API URLs and public anon keys. Preview and production can have different values.

Vibe coding

Building software by describing what you want to AI tools and iterating through prompts.

Regression break

A new change breaks something that already worked before.

Production vs preview

Preview is the builder's test environment. Production is the live app users visit.

Deployment failure

The app cannot build or launch correctly on the hosting platform.

Auth flow

The login, signup, callback, session, and redirect path that lets users access your app.

API error

A failure from a backend or external service. The browser console/network tab usually shows the useful details.
